# Kingdomino
Projet Informatique S1, DEROO Matthieu, PILI Clement

Recrée le jeu de société Kingdomino en C avec l'interface graphique SDL :)

mingw32 32bits est requis pour compiler le projet avec le la commande: mingw32-make 
Le ficher executable : ./bin/Kingdomino.exe

git pull -> met a jour le dossier

git checkout -b -> cree une branche

git add ./ -> ajoute tout les fichier ds la branche

git commit -m " " -> message pour le commit
